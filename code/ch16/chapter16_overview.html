注意力优化详解

Flash SDP 注意力

class BaselineAttention:
    # 原生注意力实现
    Q, K, V = project(x)
    attn = Q @ K.T / sqrt(d)
    attn = softmax(attn)
    return attn @ V
class FlashAttention:
    # Flash SDP 优化路径
    Q, K, V = project(x)
    return F.scaled_dot_product_attention(
        Q, K, V,
        is_causal=True
    )

块稀疏注意力

# 块稀疏模式示例
block_size = 64
sparse_mask = create_block_sparse_mask(
    seq_len,
    block_size,
    sparsity_ratio=0.5
)
# 仅计算非零块
output = flashinfer_block_sparse_attn(
    Q, K, V, sparse_mask
)
3.94x 加速: 结构化稀疏模式跳过不重要注意力块，FlashInfer内核优化，适合长序列推理

后训练量化技术

AWQ

Activation-aware Weight Quantization

  • 基于激活分布保护重要权重
  • 4-bit 权重量化
  • 保持模型精度

GPTQ

Generalized PTQ

  • 逐层量化，最小化重建误差
  • Hessian 信息引导
  • 高效量化流程

SmoothQuant

平滑量化

  • 迁移量化难度到权重
  • W8A8 混合精度
  • 数学等价变换

NVFP4

NVIDIA FP4 格式

  • Blackwell 原生支持
  • 极致压缩比
  • 硬件加速
验证方法: 使用 perplexity_eval.py 验证量化后模型精度，确保性能提升不牺牲准确性
