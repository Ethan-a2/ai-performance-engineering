Case Study: Integrated KV Cache

Baseline: NaiveKVCache

  • Per-token allocation: zeros(max_seq_len, ...)
  • Dict-based cache with allocate → append → get → free
  • Token-by-token processing loop
  • No memory reuse between requests
  • High allocation overhead per token
→

Optimized: PagedKVCache

  • Page-size slab allocation with buffer pool
  • append_block() — batch writes per block
  • Block-wise processing (block_size=8)
  • Buffer reuse via _release_buffer
  • Auto-grow with _ensure_capacity

Key Insight: The 7× speedup comes from eliminating per-token allocation and switching to block-wise processing with buffer pool reuse.

Case Study: BF16 MLP Precision Policy

Baseline: FP32 Eager

  • torch.float32 — no tensor cores
  • Separate relu() kernels (not fused)
  • Redundant normalization per forward
  • Synchronous execution
→

Optimized: BF16 Eager

  • torch.bfloat16 — tensor core acceleration
  • Same eager op graph (truthful comparison)
  • No fused kernel — pure precision policy win
  • 2.63× speedup from dtype alone
# Baseline: FP32 model
self.model = UnoptimizedModel().to(device).float().eval()

# Optimized: BF16 model (same graph, different precision)
model_dtype = torch.bfloat16 if torch.cuda.is_bf16_supported() else torch.float16
self.model = OptimizedModel().to(device, dtype=model_dtype).eval()

Case Study: Mixture-of-Experts (MoE)

Architecture

class ToyMoe(nn.Module):
    def __init__(self, hidden_dim=1024):
        self.gate = nn.Linear(hidden_dim, 2)
        self.expert0 = FFN(GELU)  # Expert 0
        self.expert1 = FFN(ReLU)  # Expert 1

    def forward(self, x):
        scores = softmax(self.gate(x))
        top_expert = argmax(scores)
        # Both experts computed, masked routing
        return out0 * mask0 + out1 * mask1

AI Kernel Generator — FlexAttention

What It Does

  • PyTorch flex_attention with custom score mods
  • Relative position encoding: score + (q_idx - kv_idx) * 1.4427
  • Causal masking via create_block_mask
  • Compiled with torch.compile(fullgraph=True)
  • Targeting Blackwell SM10x via Triton ≥ 3.5

Verification

# Compare fused vs reference attention
out_fused = flex_attention(q, k, v,
    score_mod=relative_position_score,
    block_mask=causal_mask)

out_ref = reference_attention(q, k, v,
    scale, causal=True)

max_err = (out_fused - out_ref).abs().max()
