"能否在不改变工作负载、不隐藏正确性问题的前提下，

Baseline 路径

  • Eager FP32 或最小优化的 Python 训练循环
  • 每工作项一次 CUDA kernel launch
  • 启动开销和框架开销完全可见

Optimized 路径

  • FP16 tensor-core 数学 + 融合微批次执行
  • 批量/跨步 CUDA launch 摊销调度成本

Baseline 训练循环

FP32 · 逐微批次前向+反向 · 128 个微批次 × 32 样本

# baseline_performance.py — 核心循环
def benchmark_fn(self):
    total = len(self.microbatches)
    for start in range(0, total, self.fusion):
        group = self.microbatches[start : start+self.fusion]
        group_tgt = self.targets[start : start+self.fusion]
        self.optimizer.zero_grad(set_to_none=True)
        for data, target in zip(group, group_tgt):
            logits = self.model(data)
            loss = cross_entropy(logits, target)
            (loss / group_size).backward()
        self.optimizer.step()

性能瓶颈

  • FP32 矩阵乘 — 未利用 Tensor Core
  • 逐样本启动 — 每微批次独立 kernel launch
  • 128 次前向+反向 — 启动开销累积

工作负载参数

hidden_dim = 16384

microbatch = 32 样本

num_microbatches = 128

Optimized 训练循环 FP16 + Fusion

两个优化的组合效果：精度提升 + 启动摊销

# optimized_performance.py — 核心变化
# 1. 模型转 FP16
self.model = build_training_mlp(hidden_dim)
self.model = self.model.half()  # ← Tensor Core

# 2. 预构建融合批次
self._fused_batches = []
for start in range(0, len(microbatches), fusion):
    batch = torch.cat(
        microbatches[start:start+fusion], dim=0)
    self._fused_batches.append(batch)

# 3. 更少、更大的 kernel
def benchmark_fn(self):
    for data, target in zip(
        self._fused_batches, self._fused_targets):
        self.optimizer.zero_grad(set_to_none=True)
        logits = self.model(data)
        loss = cross_entropy(logits, target)
        loss.backward()
        self.optimizer.step()

基准测试契约

每个 benchmark 必须实现的核心接口

class BaseBenchmark(ABC):
    # 必须实现 — 钩子方法
    def setup(self):         # 初始化
        raise NotImplementedError

    def benchmark_fn(self): # 被计时的函数
        raise NotImplementedError

    def teardown(self):     # 清理
        raise NotImplementedError

    # 验证接口
    def get_verify_output(self):
        raise NotImplementedError

    def get_output_tolerance(self):
        raise NotImplementedError

禁止的模式

# ❌ 常量输出 — 作弊!
return torch.tensor([0.0])

# ❌ 校验和 — 隐藏逐元素错误!
return torch.tensor([output.sum().item()])

# ❌ 静默回退 — 吞掉错误!
try: ... except: return None

正确模式

# ✅ 实际输出
return self.output.detach().clone()

# ✅ 快速失败
if self.output is None:
    raise RuntimeError("未运行")
