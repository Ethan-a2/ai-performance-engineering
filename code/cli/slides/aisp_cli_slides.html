🖥️ GPU 领域

硬件信息、拓扑、功耗、带宽分析

# GPU 信息查询
aisp gpu info
# GPU 拓扑分析
aisp gpu topology
# 功耗监控
aisp gpu power
# 带宽测试
aisp gpu bandwidth
# 硬件特性检测
aisp gpu features
✅ 支持 NVLink、PCIe、P2P 拓扑分析
✅ 实时功耗和温度监控
✅ TMA、Tensor Core、FP8 特性检测

📦 System 领域

软件栈、依赖、系统能力、环境配置

# 系统状态概览
aisp system status
# 软件版本信息
aisp system software
# 依赖健康检查
aisp system deps
# 硬件能力检测
aisp system capabilities
# 网络状态
aisp system network
# 时钟锁定验证
aisp system clock-lock-check
✅ PyTorch、CUDA、Python 版本追踪
✅ NUMA、CPU、内存层次分析
✅ 容器/cgroup 限制检测

📊 Profile 领域

性能分析、内核分析、瓶颈识别

# Nsight Systems 分析
aisp profile nsys python train.py
# Nsight Compute 内核分析
aisp profile ncu python train.py
# 火焰图生成
aisp profile flame
# 内存时间线分析
aisp profile memory
# 基线/优化对比
aisp profile compare ch07
✅ NVTX 追踪、CUDA API 分析
✅ 内核级性能指标（内存/计算/张量核心）
✅ 自动瓶颈检测和优化建议

📈 Benchmark 领域

硬件微基准测试、速度测试、诊断

# GPU 内存带宽测试
aisp benchmark memory
# PCIe 带宽测试
aisp benchmark pcie
# 张量核心吞吐量
aisp benchmark tc --precision fp16
# 综合速度测试
aisp benchmark speed --type all
# InfiniBand 带宽
aisp benchmark ib
# NCCL 集合通信
aisp benchmark nccl --collective all_reduce
✅ 内存层次结构（L1/L2/全局内存）
✅ 多精度支持（FP16/BF16/TF32/FP8）
✅ P2P/NVLink 带宽测试

🌐 Distributed & 🤖 Inference

分布式训练策略与推理优化

分布式训练

# 并行策略规划
aisp distributed plan --model-size 70 --gpus 16
# NCCL 调优
aisp distributed nccl --nodes 2 --gpus 8
# ZeRO/FSDP 配置
aisp distributed zero --model-size 70

推理优化

# vLLM 配置
aisp inference vllm --model-size 70 --gpus 4
# 量化建议
aisp inference quantize --model-size 70
# 部署配置
aisp inference serve --model llama-70b
✅ TP/PP/DP 并行策略自动规划
✅ SLURM 脚本自动生成
✅ vLLM 吞吐量/延迟估算

🧠 AI & 🏢 Cluster

AI辅助分析与集群评估

AI 助手

# 性能问答
aisp ai ask "为什么训练这么慢？"
# 概念解释
aisp ai explain "Tensor并行"
# 故障诊断
aisp ai troubleshoot "CUDA OOM"

集群评估

# 评估套件
aisp cluster eval-suite --mode full --hosts node1,node2
# 通用评估
aisp cluster common-eval --preset core-system --hosts node1
# 字段报告验证
aisp cluster validate-field-report --report field-report.md
✅ LLM 驱动的性能分析
✅ 书籍引用和最佳实践
✅ 集群级基准测试和报告生成
