Key Principle

Every run produces a self-contained artifact directory:

cluster/runs/<run_id>/
  manifest.json
  structured/   ← machine-readable results
  raw/          ← raw logs
  figures/      ← generated plots
  reports/      ← rendered reports

The run directory is the unit of record — not ad-hoc files.
